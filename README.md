# RAYLIB STARTER PACK (VERSION C23)

Welcome to the Raylib Starter Pack (version C23)! This project provides a foundational framework for building 2D games using the Raylib library in C23. It includes essential components such as entity management, memory management, and screen handling to help you get started quickly.

## Features
- Basic 2D game framework structure
- Entity management system
- Memory management utilities
- Screen handling for different game states (e.g., title screen, gameplay, options)
- Integration with Raylib for graphics and input handling 
- Audio management support
- Sample assets and fonts included

## Getting Started
1. **Clone the Repository**:
    ```bash
    git clone https://github.com/Dubnium32x/raylib-starter-pack.git
    cd raylib-starter-pack
    ```
2. **Install Raylib**:
    Follow the instructions on the [Raylib GitHub page](https://github.com/raysan5/raylib) to install Raylib on your system.
3. **Build the Project**:
    Use your preferred C build system (Makefile, CMake, etc.) to compile the project. Ensure that Raylib is correctly linked.
    ```bash
    make check-raylib
    make
    ```
4. **Run the Game**:
    Execute the compiled binary to launch the starter pack game framework.
    ```bash
    ./bin/raylib-starter-pack
    ```

## Project Structure
- `src/`: Contains the source code for the project.
- `res/`: Contains assets such as images, audio, and fonts.
- `obj/`: Directory for compiled object files.
- `bin/`: Directory for the final executable binary.
